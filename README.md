## ✨tyutils

**tyutils** 是一个现代化的 JavaScript / TypeScript 工具库，专注于提升前端开发效率。
它提供了从 **虚拟列表渲染**、**异步任务调度** 到 **大文件上传** 等实用功能，帮助你在复杂业务中快速构建高性能、高可维护的前端模块。

未来还将支持：

- 通用数据结构与算法工具（如 LRU 缓存、任务队列、优先级调度器）
- DOM 与性能优化工具（节流、防抖、IntersectionObserver 封装等）
- 网络与存储模块（请求中间层、IndexedDB 封装、本地缓存管理）
- 智能工具（任务并发控制、重试机制、文件分片校验）
